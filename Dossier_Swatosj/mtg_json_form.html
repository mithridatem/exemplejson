MTGJSON Database v1

Fonctionnalités de cette version

  • Récupère les données des cartes dans la base de données JSON d'un format d'extension donné (Modern, Legacy...)
  • Enregistre les informations des cartes dans la base de données locale cartes (configuré au préalable)
  • Le nombre d'extensions sauvegardées et le nombre de cartes traités par chaque INSERT sont paramétrables
  • Affichage du temps de la procédure et de la moyenne par extension
